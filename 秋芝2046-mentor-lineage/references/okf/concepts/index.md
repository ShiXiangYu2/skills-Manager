# Concepts

- [Agent智能体](001-agent智能体.md) - 能够自主执行任务的软件系统，通过自动化流程提高效率。
- [RAG](002-rag.md) - Retrieval-Augmented Generation，一种结合检索和生成模型的技术，用于增强AI系统的知识获取能力。
- [API](003-api.md) - Application Programming Interface，软件之间的通信接口，允许不同的应用程序互相交互。
- [Codex](004-codex.md) - 一个由OpenAI开发的编程助手，能够帮助开发者自动完成代码编写任务。
- [Grok](005-grok.md) - 一种强大的AI模型，用于自然语言处理和生成。
- [GPT](006-gpt.md) - Generative Pre-trained Transformer，一种广泛使用的语言模型，能够生成自然语言文本。
- [DeepSeek](007-deepseek.md) - 一款结合了多种AI技术的搜索工具，能够提供更精准的搜索结果。
- [NotebookLM](008-notebooklm.md) - 一个基于AI的文本生成工具，特别适用于撰写复杂的文案。
- [Flux Kontext](009-flux-kontext.md) - 一个能够通过简短的文本输入快速生成图片的AI工具。
- [Token](010-token.md) - 在AI模型中，文本的基本单位，通常是一个单词或字符。
- [ID](011-id.md) - 识别特定资源或用户的唯一标识符。
- [Workflow](012-workflow.md) - 工作流程，指完成特定任务的一系列步骤。
- [提示词](013-提示词.md) - 在生成模型中，用于引导模型生成特定内容的输入。
- [程序接口](014-程序接口.md) - 软件系统中供其他程序调用的接口。
- [知识库](015-知识库.md) - 存储和管理知识的系统，常用于提高AI模型的性能。
- [数字员工](016-数字员工.md) - 通过AI技术模拟的虚拟员工，能够执行多种任务。
- [场景行动](017-场景行动.md) - 特定场景下的具体操作步骤。
- [伦理机制](018-伦理机制.md) - 确保AI技术应用符合伦理标准的机制。
- [后果](019-后果.md) - AI技术应用的结果和潜在影响。
- [运气](020-运气.md) - 在某些AI应用中，结果可能受到不可预测因素的影响。
