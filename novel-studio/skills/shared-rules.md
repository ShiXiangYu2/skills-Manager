# 共享规则

> 本文件包含所有子技能共享的规则，避免重复定义。

## 一、章节字数标准

每章字数以番茄小说后台/番茄字数统计口径为准。扩写目标是"4500字以上即可"，不要为了追求更高字数强行灌水。

**如果正文已经达到 4500 字以上：**
1. 允许自然收束，不需要继续扩写到 5000、6000 或更长。
2. 不要增加重复复盘、重复心理活动、重复系统面板、重复目标宣言来凑字数。
3. 后续新增内容必须带来新行动、新信息、新冲突或新后果。
4. 如果只能靠重复前文来增加字数，必须停止扩写，改为收尾。

**字数不足 4500 字时，优先扩写：**
1. 场景动作和人物互动。
2. 冲突升级和后果呈现。
3. 角色做选择的过程。
4. 与主线相关的新线索。

**不要用以下方式补字数：**
1. 大段回顾前文。
2. 反复解释人物已经明白的道理。
3. 重复"他需要更多证据/盟友/勇气"。
4. 重复"沉默几秒/心跳漏拍/心里一紧"等模板反应。
5. 把同一场景换说法再写一遍。

## 二、禁止的AI味模板反应句

以下句式具有明显 AI 扩写味，本章每类最多出现一次，能不用就不用：

| 禁止句式 | 处理方式 |
|----------|----------|
| "心跳漏了一拍" | 删除或改成具体动作 |
| "沉默了X秒" | 删除或改成场景停顿 |
| "心里一紧" | 删除或改成感官细节 |
| "瞳孔微缩" | 删除或改成具体反应 |
| "嘴角抽了抽" | 删除或改成选择性表情 |
| "喉咙发干" | 删除或改成吞咽动作 |
| "后背发凉" | 删除或改成环境描写 |
| "脑子嗡的一声" | 删除或改成思维中断 |
| "空气安静下来" | 删除或改成具体声音消失 |
| "没有立刻接话" | 删除或改成场景专属动作 |

**处理顺序：**
1. 先判断这句是否有剧情功能。
2. 如果删掉不影响剧情、信息、人物关系，直接删除。
3. 如果必须保留停顿，改成场景专属动作或对话节奏。
4. 如果必须保留情绪，必须绑定当前物件、危险、人物习惯或下一步选择。

**禁止"模板替换模板"：** 不允许把"沉默了三秒"机械替换成"没有立刻接话"。每一次改写都必须从当前场景里取材料。

## 三、每一段必须有新功能

正文中每个大段落必须至少满足以下一种功能：
1. **新行动**：角色做了新的事。
2. **新信息**：读者获得未出现过的信息。
3. **新冲突**：人物关系、危险、选择出现变化。
4. **新后果**：之前行为导致新的代价或收益。
5. **新情绪层次**：不是重复"震惊/沉默/下定决心"，而是更具体的反应。

**如果一个段落只是重复说明"证据还不够""沃特很危险""他需要盟友""热度还在跌"，必须删除或压缩成一句。**

**如果一个段落只是写角色"心跳漏拍、沉默几秒、心里一紧、嘴角抽动"，但没有带来新行动或新信息，必须改写。**

## 四、章节末尾规则

章节结尾可以升华，但只能升华一次。

**结尾优先写"新的危险/新的选择/新的行动钩子"，不要反复总结主角目标。**

**推荐结尾结构：**
1. 当前行动结束。
2. 新问题露出。
3. 主角做出一个具体决定。
4. 留一个下一章能接住的钩子。

**不要使用：**
> 保存证据。
> 揭露真相。
> 让沃特付出代价。
> 而要做到这些，他需要……

这种列表式终章宣言，除非这是全章唯一一次总结。
