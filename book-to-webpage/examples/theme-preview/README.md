# 主题预览（4 个风格对照）

同一份内容（土地财政主题，覆盖 Hero / Tab / 因果链 / 类型选择器 / Accordion / 时间线 / 前后对比 / 对比矩阵 / 案例卡 / 引用卡 / 问题清单）用 4 个主题各渲一版，浏览器直接打开对照观感：

| 文件 | 主题 | 一句话 |
|---|---|---|
| `warm-paper.html` | 暖纸典藏（默认） | 暖纸色 + 宋体 + 金红蓝绿 + 金色光晕 |
| `minimal.html` | 极简学术 | 纯白留白 + 无衬线 + 单一岩蓝 |
| `dark.html` | 夜读深空 | 深墨背景 + 暖白文字 + 柔金（护眼夜读）|
| `ink-wash.html` | 东方水墨 | 宣纸白 + 水墨黑 + 朱砂红 + 「」书名号 |

> 内容为示意（页码出处非真实），仅用于展示主题差异。真实页面由 skill 据知识库生成。

主题定义见 `templates/themes/`，目录与设计要点见 `templates/themes/README.md`。
