# 风格主题目录（单一真相源）

生成网页时，Agent 读取用户选中的主题文件，把内容整段注入 `base.html` 的 `{{theme_style}}` 槽位（紧跟 `:root` 默认值之后，故覆盖生效），并在 `<body>` 上设 `data-theme="{{theme_name}}"`。

**布局/交互零改动**——主题只覆盖 CSS 变量（字体/底色/强调色/纸面 tint）+ 少量 `[data-theme="x"]` 组件微调。

## 主题清单（菜单用）

| `theme_name` | 文件 | 中文名 | 一句话 |
|---|---|---|---|
| `warm-paper` | `warm-paper.css` | 暖纸典藏（默认） | 暖纸色纸面 + 宋体正文 + 金红蓝绿点缀 + 金色径向光晕，温润书卷气 |
| `minimal` | `minimal.css` | 极简学术 | 纯白留白 + 无衬线（苹方）+ 近单色 + 单一克制岩蓝强调，瑞士国际主义 |
| `dark` | `dark.css` | 夜读深空 | 深墨背景 + 暖白文字 + 柔金点缀，护眼夜读 |
| `ink-wash` | `ink-wash.css` | 东方水墨 | 宣纸白底 + 水墨黑字 + 朱砂印章红单点 + 楷宋书法味，中式人文 |
| `vintage-editorial` | `vintage-editorial.css` | 复古编辑 | 奶油净底 + 无衬线正文 + 砖红强调 + ▸/▾ accordion + ❝ 引号，英式杂志感 |
| `paper-ink` | `paper-ink.css` | 纸墨风 | 暖米纸底 + 楷体正文 + 猩红强调 + ❧/❝ 西式装饰，文学杂志气质 |

## 设计要点

- **暖纸典藏**：base.html `:root` 默认值即此主题；本文件是其显式拷贝，保证菜单 6 项对称。
- **极简学术**：核心差异是正文换无衬线（苹方）、底色去渐变改纯白、把金红蓝绿四色收敛到「墨 + 单一岩蓝」，整体去彩虹、留白大。
- **夜读深空**：`--ink/--white` 在深色下反相（原本浅底深字 → 深底浅字）。`--white` 被复用为「卡片底」+「落在 ink 上的反相文字」，深色会撞车，故用 `[data-theme="dark"]` 局部覆盖 selected tab / tooltip / toast / 卡片底。
- **东方水墨**：收敛到「墨黑 + 朱砂单红」，标题去黑体改宋/楷统一，accordion 标记用「」书名号替代 +/−，引号用『』，留白加大。
- **复古编辑**：净奶油底（去渐变） + 正文换无衬线（苹方）+ 砖红单点，▸/▾ 三角 accordion + ❝ 引号装饰，英式杂志/editorial 味。
- **纸墨风**：暖米纸底 + 楷体正文 + 猩红强调，❧/❝ 西式装饰，西式 literary magazine 风格；区别于 ink-wash 的中式宣纸/朱砂/「」。

## 新增主题

复制任一 `.css` 改名，重写 `:root` 变量 + 必要的 `[data-theme]` 覆盖，再在本表加一行 + 更新 `SKILL.md` 的菜单表即可。
