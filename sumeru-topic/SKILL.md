## 网文选题策划

### 触发关键词
想写小说不知道写什么、帮我想个小说题材、找网文创意、分析什么题材火、小说选题评估、现在什么网文题材好卖、给我几个小说创意、想个小说金手指、帮我找个热门题材、小说写什么题材容易火、网文市场趋势分析、帮我做小说选题、网文热门题材分析

### 核心功能
1. **拆书训练（前置步骤）**：分析素材库中的成功案例，提取可迁移结构、爽点模式、人物模板和技法
2. **市场热点分析**：基于AI推理分析网文市场热点题材、流行趋势与受众偏好（注：市场分析基于AI对网文领域的知识推理，非实时数据，建议结合实际平台榜单交叉验证）
3. **多维度选题生成**：结合输入的题材类型和关键词，生成3-5个具有市场潜力的选题方向
4. **核心创意设计**：为每个选题设计金手指、核心设定、爽点模式、情绪价值曲线
5. **选题可行性评估**：从市场竞争、受众规模、创作难度、变现潜力、平台适配性等维度进行综合评估
6. **创作方向建议**：提供开篇切入点、故事节奏规划、人物设定框架、差异化竞争策略
7. **风险提示**：识别选题可能面临的政策风险、市场饱和风险、题材生命周期等问题

### 3.3 最终推荐与理由

### 3.4 平台适配性评估（必填）

**核心原则：学起点，发番茄**
- 起点优势：精品多、成熟作者多、类型迭代快，适合学习结构、节奏、人物塑造和商业写法
- 番茄优势：新人更容易上场，推荐土壤相对友好，更适合新手试错、验证题材和训练稳定更新能力

**当前策略**：
1. 在起点学习头部与腰部作品的写法
2. 将拆解成果转化为自己的故事能力
3. 优先在番茄小说发布和测试
4. 如果一本书不合适，写到15万字以后及时止损

#### 平台选择评估表

| 评估维度 | 起点中文网 | 番茄小说 | 建议 |
|----------|-----------|---------|------|
| **读者口味** | 偏好精品、有深度、文笔好 | 偏好快节奏、爽点密、代入感强 | 根据题材选择 |
| **字数要求** | 3000-5000字/章 | 2000-3000字/章 | 番茄适合短章节 |
| **完读率标准** | 较高（读者耐心强） | 较低（前3章定生死） | 番茄开篇必须快 |
| **推荐机制** | 编辑推荐+算法 | 纯算法推荐 | 番茄更依赖数据 |
| **新人友好度** | 较低（头部效应强） | 较高（新书有流量扶持） | 新人优先番茄 |
| **签约门槛** | 较高（需要一定字数和质量） | 较低（数据好即可签约） | 番茄更容易起步 |
| **收入模式** | 订阅+打赏 | 广告分成+付费 | 番茄靠量取胜 |
| **IP改编潜力** | 较高 | 中等 | 起点更适合长线 |

#### 选题-平台匹配建议

根据选题特征，推荐发布平台：

| 选题特征 | 推荐平台 | 理由 |
|----------|---------|------|
| 高武玄幻、都市异能 | 起点 | 读者基础好，付费意愿强 |
| 甜宠言情、先婚后爱 | 番茄 | 女性用户多，快节奏受欢迎 |
| 悬疑推理、无限流 | 起点 | 需要深度阅读，番茄完读率低 |
| 系统流、重生、神豪 | 番茄 | 爽点密集，适合短章节 |
| 历史权谋、种田基建 | 起点 | 需要耐心阅读，番茄不适合 |
| 脑洞文、搞笑文 | 番茄 | 轻松娱乐，传播性强 |

#### 番茄试水建议

如果选择在番茄试水：
1. **前3章必须抓人**：前1000字出现处境+冲突+金手指
2. **章节字数控制**：2000-3000字/章，不要写太长
3. **爽点密度**：每章至少1个爽点，不能有"水章"
4. **章末钩子**：每章结尾必须有具体悬念
5. **更新频率**：日更2-3章，保持稳定
6. **15万字止损**：写到15万字后判断数据，不适合就切书

#### 起点学习建议

如果选择在起点学习：
1. **拆书重点**：前100章，学习开篇、节奏、爽点循环
2. **关注榜单**：新书榜、畅销榜、分类榜
3. **分析评论区**：读者反馈集中在哪些点
4. **提炼可迁移结构**：不要照搬内容，要学习写法
