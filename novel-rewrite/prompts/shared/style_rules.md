# 写作风格规则

## 去重规则（最高优先级）

每段必须承担明确功能，至少满足一种：
1. 推进情节
2. 揭示人物状态
3. 加深冲突
4. 回收伏笔
5. 建立下一章钩子

禁止以下重复：
- 用不同句式重复同一情绪
- 重复说明角色已经知道的信息
- 重复描写同一种疼痛、震惊、沉默、凝视
- 重复使用"他意识到/某种感觉蔓延/空气安静下来"等模板句
- 重复前文已经完成的心理转折

## 禁止的AI味模板句

| 禁止句式 | 替换方向 |
|----------|----------|
| "心跳漏了一拍" | 具体动作：手停在半空、脚步顿了一下 |
| "沉默了X秒" | 场景停顿：屏幕亮着没人先碰它 |
| "心里一紧" | 感官细节：胃缩了一下、后颈发麻 |
| "瞳孔微缩" | 具体反应：眼睛眯了一下、盯着某个细节不动了 |
| "嘴角抽了抽" | 选择性表情：没接话、把脸转向另一边 |
| "喉咙发干" | 吞咽动作、喝水、清嗓子 |
| "后背发凉" | 环境描写：风从巷口灌进来 |
| "脑子嗡的一声" | 思维中断：话说到一半停住了 |
| "空气安静下来" | 具体声音消失：收音机还在响但没人说话 |
| "没有立刻接话" | 删除停顿或改成场景专属动作 |
| "手指不自觉地收紧" | 具体物件：把手机倒扣过去 |

同一章每类最多出现一次，能不用就不用。不要"模板替换模板"。

## 字数规则

- 目标字数：4500-6000字
- 超过4500字后自然收束，不灌水
- 不要为了凑字数添加重复内容

## 章末钩子规则

- 结尾必须有下一章必须处理的问题
- 不要用"他不知道的是"等旁白钩子
- 用具体动作、消息、人物选择作为钩子

## 格式规范

- 暗影内心声音用【...】方括号
- 分节用 --- 分割线
- 对话用引号
- 不要使用游戏系统面板格式
